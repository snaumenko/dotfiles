# dotfiles

## Prerequisites
0. brew
1. install zsh
2. install oh-my-zsh
3. install autojump `brew install autojump`
4. install font `brew install --cask font-jetbrains-mono-nerd-font` and set it in the terminal app

1. Clone the repo in $HOME/dotfiles
2. `cd $HOME/dotfiles`
3. `stow zsh`
4. `stow git`
5. `stow cursor`
